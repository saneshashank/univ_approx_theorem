Universal Approximation Playground — Deep

Add up to 10 hidden layers of random features and fit a linear readout via ridge regression (closed form). Adjust depth/width to see approximation improve.

UAT: a single hidden layer with a non-polynomial activation can approximate any continuous function on a compact domain. Discontinuous targets won’t be uniformly approximated, but MSE can still drop.

Depth: 1 / 10

Each layer width: 1–2048. Drag sliders to set neurons per layer.
